Berita Pengingatan: Penutupan Server Maintenance Rizz Store

Diberitahukan kepada seluruh pelanggan setia Rizz Store bahwa kami akan melakukan penutupan sementara server untuk kegiatan maintenance yang diperlukan guna meningkatkan kualitas layanan kami. Penutupan server ini dijadwalkan akan dilakukan pada hari ini, mulai pukul 08.00 hingga pukul 16.00. Selama periode penutupan server ini, layanan pembelian dan transaksi online akan ditangguhkan. Hal ini berarti Anda mungkin tidak dapat mengakses situs web atau aplikasi kami untuk melakukan pembelian atau mendapatkan informasi produk. Kami memohon maaf atas ketidaknyamanan yang mungkin timbul akibat penutupan sementara ini. Namun, langkah ini kami ambil untuk memastikan bahwa layanan kami tetap optimal dan berkualitas bagi Anda. Setelah selesai dilakukan maintenance, kami akan segera mengaktifkan kembali layanan kami agar Anda dapat melanjutkan pengalaman berbelanja yang menyenangkan di Rizz Store. Terima kasih atas pengertian dan dukungan Anda selama ini. Kami berharap Anda tetap setia bersama kami di Rizz Store. Untuk informasi lebih lanjut, jangan ragu untuk menghubungi layanan pelanggan kami.
